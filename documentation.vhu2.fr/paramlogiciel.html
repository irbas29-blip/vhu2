Paramétrage Logiciel et Interfaces

Lors de l'installation de votre logiciel, il est très important que vous le paramétriez le plus possible.

Pour cela, rendez vous dans la section Paramétrages du logiciel, puis dans la section Site. Commencez par remplir les informations sur votre entreprise en haut et à gauche de la page.



Le reste de la page est composé de multiples onglets. Les premiers permettent d'entrer plusieurs informations qui seront réutilisées notamment dans les documents.



Sur l'exemple ci-dessus, l'entête de toutes les factures, bons de livraison etc.. est détaillé. Il apparaitra donc automatiquement sur ceux-ci.

Dans l'onglet dessous, vous pouvez rédiger les écritures présentes en fin de document pour chaque type de document.



Vous devriez aussi insérer votre logo dans l'onglet dédié.

Les onglets de Facturation jusqu'à Options GC sont des vrais paramètres à proprement parler : ils permettent de régler de nombreux détails pour adapter votre logiciel à vos méthodes de travail.
Par exemple la case ci-dessous permet de ne pas montrer dans votre catalogue les pièces dont le stock est nul. Cela permet une meilleur visibilté de vos stocks.



La case ci-dessous permet d'afficher les numéros de livre de police sur la facture.



Comme vous pouvez le constater, les options sont très nombreuses alors n'hésitez pas à jeter un coup d'oeil !

Si vous cherchez quelque chose de particulier n'hésitez pas à contacter notre SAV.

Si vous avez un souci d'ergonomie avec le logiciel, il y a de grandes chances que l'on puisse le régler en cochant une simple case !
