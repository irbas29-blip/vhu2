Sortir un Lot

Si un client achète plusieurs véhicules ou que vous vous débarassez de plusieurs VHU d'une autre manière, vous avez la possibilité de les sortir de votre parc par lot.

Pour cela, rendez vous dans la section Entrées/sorties en lot puis dans Sortie de véhicules par lot.



Pour sélectionner les véhicules, vous avez deux possibilités :

- Si vos véhicules étaient déjà entrés par lot, vous pouvez rechercher ce lot avec la loupe, sur la droite de l'écran.

- Sinon, vous pouvez cliquer sur Liste des VHU sur parc pour accéder à tout le parc.



Il suffit alors de cocher les véhicules qui vous intéressent avec les cases à gauche.



Cliquez en bas à droite sur la case de validation et cliquez ensuite sur Ajouter / Modifier, vos véhicules sont sélectionnés.
Remplissez alors la zone inférieure avec les informations concernant le lot dans sa globalité.


Vous aurez également la possibilité de vous occuper de toutes les démarches SIV.

Cliquez sur Facture pour être redirigé vers la facturation véhicule.

Après avoir fini la facturation, il ne reste plus qu'à cliquer sur Sortie du parc.
