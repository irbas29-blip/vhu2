Global PRE

Notre partenaire Global PRE vous offre plusieurs avantages en cas d'abonnement, comme par exemple des fiches de démontage adaptées à chaque type de véhicule pour toujours avoir une liste exhaustive des pièces récupérables sur un véhicule.

Si cela vous intéresse, veuillez contacter notre SAV.

Comment utiliser l'interface Global PRE ?

Vous pouvez accéder à cette interface de fiches de démontage Global PRE de plusieurs façons :

- via l'outil des fiches de démontage

- via la fiche d'un véhicule.

Via l'outil des fiches de démontage

Une fois votre abonnement à ce service effectué, vous pouvez accéder à l'interface Global PRE depuis votre logiciel VHU2 en suivant le chemin ci-dessous :

Sur le menu principal, cliquez sur Gestion des articles puis sur l'onglet Fiches de démontage.



Dans le menu des fiches de démontage, cliquez sur Nouvelle Fiche de Démontage. Cela vous affichera un message vous demandant si vous voulez créer une fiche de démontage avec les informations de Global PRE.



Après avoir accepté le message précédent, vous arriverez sur une nouvelle fenêtre.

Saisissez par exemple l'immatriculation du véhicule dont vous souhaitez obtenir la fiche de démontage puis cliquez sur le bouton Obtenir le véhicule. Les champs des informations concernant le véhicule se rempliront alors automatiquement.



Une fenêtre apparaît pour vous demander de choisir la bonne version du véhicule.



Si vous ne savez pas, cliquez sur le bouton Sélection à partir des équipements. Vous pourrez alors identifier la version du véhicule à partir des équipements qu'il possède.



Après avoir sélectionné la bonne version du véhicule, un message vous demandera si vous souhaitez obtenir les pièces du véhicule. En acceptant, le tableau en fond se chargera avec l'ensemble des pièces du véhicule regroupés dans des familles.

Naviguez dans l'arborescence pour trouver et sélectionner les pièces à démonter qui vous intéressent pour les rentrer au stock par la suite.

Via la fiche d'un véhicule

Vous pouvez gérer la fiche de démontage du véhicule de votre choix directement depuis la fiche de ce véhicule en procédant comme suit ou via le menu rapide des derniers dossiers :



A partir de là, vous pouvez rechercher le véhicule avec son immatriculation.

Vous arriverez ensuite sur la fiche du véhicule où vous pourrez retrouver les fiches de démontage dans l'onglet Démontage.



Cela vous amènera dans la fenêtre des fiches de démontage d'où vous pourrez créer une nouvelle fiche pour ce véhicule en cliquant sur le bouton en bas à gauche.



VHU 2 vous demandera ainsi si vous voulez créer cette fiche avec les données et les informations fournies par Global PRE.



Les données du véhicule sélectionné seront automatiquement enregistrées et vous n'avez plus qu'à cliquer sur le bouton Obtenir le véhicule.



Il ne vous reste plus qu'à sélectionner la bonne version du véhicule en vous aidant si besoin de la fenêtre des équipements embarqués.





Après avoir sélectionné la bonne version du véhicule, un message vous demandera si vous souhaitez obtenir les pièces du véhicule. En acceptant, le tableau en fond se chargera avec l'ensemble des pièces du véhicule regroupés dans des familles.

Naviguez dans l'arborescence pour trouver et sélectionner les pièces à démonter qui vous intéressent pour les rentrer au stock par la suite.
