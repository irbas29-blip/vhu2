Vos Statistiques

VHU2 vous offre la possibilité de consulter de nombreuses statistiques pouvant s'avérer utiles au sujet de votre activité.

Pour les consulter, cliquez simplement sur Statistiques depuis le menu principal.


Dans la plupart des sections, le fonctionnement reste le même. Nous allons l'illustrer avec le Hit-parade des ventes de véhicules :


La première chose à faire pour toutes les sections sera de renseigner la période concernée.

Ensuite, il faudra remplir d'autres champs qui dépendront d'une statistique à une autre.

Ici, il faudra par exemple choisir selon quels critères on cherche les meilleurs éléments concernant les ventes de véhicules (meilleurs marques, meilleurs acheteurs etc...). Ces infos sont suffisantes, vous pouvez cliquer sur Rechercher.


Vous obtenez alors une étude poussée sur les marques les plus rentables, qui représentent le plus de ventes ou qui représentent le plus de chiffre d'affaires.

Les autre critères permettent d'affiner la recherche, en sélectionnant par exemple un garage dans l'onglet Acheteur, nous aurons le classement des meilleures ventes par marque à ce client.

On comprend avec ces exemples qu'ont peut avoir des statistiques très détaillées qui correspondent parfaitement à ce que vous voulez savoir.
